Exercise 1

Starting to program a simple game

Background

In the last exercise, you wrote some text. In this exercise, we'll be transforming that text into code.

Instructions

  1. Download this starter kit. Take a look around and make sure you understand the file structure.
  2. Take a look at src.js. Note that there will be a global Object called player that contains the current position as x and y properties: 
    player.x = 0; // sets the player's x posiiton to 0
player.y = 2; // sets the player's y position to 2
  3. Start with moveRight
  4. In spec.js, add tests that equate to the descriptions you wrote.
  5. In src.js, write code for the functions that you are testing. All functions should go in the global scope.
  6. Repeat for moveLeft, moveDown, and moveUp
