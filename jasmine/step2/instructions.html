Exercise 2

Checking who won

Background

In the last exercise, you wrote some text. In this exercise, we'll be transforming that text into code.

Instructions

  1. There will be a global Object called goal that contains the current position of the goal as x and y properties: 
    console.log(goal.x); // logs 0
console.log(goal.y); // logs 0
  2. There will be a function called checkWin that checks whether the player has won. checkWin will return a true if the player's x and y coordinates match those of the goal. Otherwise, checkWin will return false.
  3. checkWin should be called whenever moveRight, moveLeft, moveDown, or moveUp are called.
  4. In GameSpec.js, write tests for the checkWin function. Remember that you sometimes have to "set the stage" before running the test. You should also be sure to use some spies.
  5. In Game.js, write code for checkWin.
