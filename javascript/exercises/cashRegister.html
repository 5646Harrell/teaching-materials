The Cash Register

  • Write a function called cashRegister that will take one argument (an array of a strings) and returns the total. Note, you will have to convert the string to a number value to compute the total. 

    function cashRegister(arr) {
    var total = 0;   

    for(var i = 0; i < arr.length; i++){
      var price = arr[i].substr(1);
      total += Number(price);
    }
    return total;
}

cashRegister(["$23.04", "$432.25", "$8.07"]);
