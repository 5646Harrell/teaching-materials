The Cash Register

  • Write a function called cashRegister that takes an object containing shopping cart values for various customers. The function should return an array that contains the name of the person and the sum of their cart. (ie [ [ 'Ernie', 237.54 ], [ 'Bert', 3.38 ] ])''
  • Example input 
    var prices = {
  brenda: { 
    banana: "$1.25",
    handkerchief: "$.99",
    Tshirt: "$25.01"
    },
  claire: {
    apple: "$0.60",
    nalgene: "$10.34",
    proteinShake: "$22.36"
    }
};

// Solution

// [ "brenda": 33.3, "claire": 33.3 ]
