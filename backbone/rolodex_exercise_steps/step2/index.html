Exercise 3 - Making a View

In this exercise, we will be creating a View. We will be instantiating it with our previous model.

Instructions

  1. Download this file
  2. Open SpecRunner.html in a browser
  3. In src/app.js, define a variable called "View". Write code to make all the tests pass.

Bonus - Changing it up

  1. Create an .html file
  2. Load in the necessary libraries, either from a local file system or cdn (don't forget about dependencies!)
  3. In the <body> tag, create a DOM node that you'll refer to later, like <div id="thumb-anhcor"></div>
  4. In src/app.js, change your View so that it refers to the existing element with el instead of className and tagName
  5. Use jQuery to create an img tag with a src attribute like before
  6. When the view renders, append that img node to the $el

Double Bonus - Do something fancy

  1. Do something fancy with that onClick event, and maybe add some more events on top of that!
