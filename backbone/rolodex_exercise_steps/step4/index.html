Exercise 5 - Fetching and rendering a Collection

In this creative exercise, we will be putting all of our skills from the day together. Feel free to follow the instructions below, or branch out to practice skills at your own pace!

Instructions - Fetching the Collection

  1. Set up an HTML page with the libraries and dependencies for Backbone. Use this page as an example.
  2. Define a Collection whose url property is "http://spacodemo.herokuapp.com/products"
  3. Have the Collection call .fetch() when it instantiates

Instructions - Parsing the Models

  1. There are some data inconsistencies with our models. Use model.parse() to fix some of them (like pricing)
  2. Add some additional data processing for each model. For example, if there are items in stock, set 'inStock' to true

Instructions - Creating a View

  1. Create a View with a Template! You can use Handlebars templates like the last exercise, or find another way to generate templates

Instructions - $(document).ready()

  1. When the document is ready, instantiate your Collection and View
  2. When the document is ready, append your View to the DOM
