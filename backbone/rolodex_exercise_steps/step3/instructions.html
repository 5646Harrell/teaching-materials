Exercise 4 - Rendering a Template in a Collection

In this exercise, we will be using Handlebars templates. We'll also render them with a collection.

  • If you didn't finish the last exercise, download the solution files from here.
  • Continue from your previous files or the solution files.

Part 1

  1. Open index.html and find the template with id rolodex-template.
  2. Add an #each Handlebars block helper
  3. Inside of the #each helper, add Handlebars {{expressions}}.
    • Later, you will use JSON to render this template, so make sure that your attributes match. Examples from the models we just made are {{firstName}} and {{lastName}}. Feel free to stick to the attributes given, or switch it up.

Part 2

In the last exercise, we rendered a View of a Model. This time, we will render a View of a Collection.

  1. Open SpecRunner.html in a browser
  2. Click on Spec List on the upper-left hand corner.
  3. Make changes in src/app.js to make all the tests for Exercise 4 pass.
  4. Inside of your .render function, compile the Handlebars template, pass in JSON from the collection, and append it to the view's $el
  5. Don't forget to return this at the end of .render
  6. In the .initialize function, use .on() or .listenTo() to add an event listener for changes to the collection. The callback should be .render
  7. Update the $(document).ready() callback from the last exercise. Instead of rendering and appending your personView to the DOM, render and append your rolodexView.

Bonus - Loop it different ways!

In the above exercises, we used a for loop within a Handlebars #each block. Try moving the loop into the .render() function of the view.
