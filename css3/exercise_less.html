CSS: Less Exercise

Note: You will have to use Firefox for this exercise, as Chrome does not work with Less when you're developing on the file system.

  1. Download this webpage into a folder. You'll be turning the CSS in that file into Less instead.
  2. Open the webpage and see what it looks like.
  3. Download less.js from lesscss.org and put it in the same folder as your exercise.
  4. Move the styles from the webpage into a separate file and save it as "styles.less".
  5. Add links to the LESS and JS file in the HEAD of your page: 
    <link rel="stylesheet/less" type="text/css" href="styles.less">
<script src="http://cloud.github.com/downloads/cloudhead/less.js/less-1.3.1.min.js" type="text/javascript"></script>
  6. Open the page in Firefox and check that it loads and is styled.
  7. Find all the colors used in the styles and turn them into variables. Less docs: variables
  8. Create a mix-in for rounded-corners and use that wherever border-radius is used. Less docs: mix-ins
  9. Create a mix-in for box-shadow and use that wherever box-shadow is used. Less docs: mix-ins (scroll)
  10. Re-structure the styles so that they are nested according to the DOM structure. For example, all of the sidebar related styles can nest underneath a main sidebar selector. Less docs: nested rules
